このサイトをリニューアルしました。

## なぜリニューアルしたのか

もともとこのサイトは1年生のときにぱぱっと作ったもので、更新が面倒で全然更新しなかったりといろいろ問題がありました。  
なので、markdownで書いて簡単に更新できるようにしました。これで更新に対するハードルが下がれば嬉しいな(更新するとは言ってない)

## どうやって作ったのか

上記理由でmdxが扱えると嬉しくて、更になんか最近使えるようになったらしい View Transitions API でぎゅい～んって動かしてみたかったので Astro を採用してみました。  
Bun とか Panda CSS とか使ってるのはとりあえずこういうサイトを作るときには新しいもの使ってキャッチアップしとけの精神です。
